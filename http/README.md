# HTTP

A set of types for representing HTTP requests and responses.

## Usage

To use `http`, first add this to your `Cargo.toml`:

```toml
[dependencies]
http = "0.1"
```

Next, add this to your crate:

```rust
extern crate http;

use http::{Request, Response};
```
